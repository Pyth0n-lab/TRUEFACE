# TRUEFACE 

Программное обеспечение на базе искусственного интеллекта для распознавания и верификации лиц. Проект успешно прошёл валидацию и защиту перед экспертной комиссией. MVP (минимально жизнеспособный продукт) софта был спроектирован и собран в сжатые сроки (за 24 часа).

## 🛠 Технологический стек
* **Language:** Python
* **ML/DL Frameworks:** Keras / TensorFlow (используется предобученная нейросетевая модель `.h5`)
* **GUI Framework:** [Впиши сюда библиотеку интерфейса, например: Tkinter / PyQt / CustomTkinter]
* **Environment:** PyCharm, Git

## 🚀 Архитектура проекта
* `main.py` — главный файл инициализации и запуска приложения.
* `detector.py` — логика работы нейросети, обработка входных кадров и распознавание лиц.
* `gui.py` — графический интерфейс пользователя (GUI) для взаимодействия с системой.
* `test.html` — инструмент для локального тестирования и валидации работы алгоритмов.
